VALUE FORMULA
Value Source
The data source used to value players everywhere — trade calc, league syncs, rankings, and scouts. KTC Dynasty is the default.
Scaled Power Curve
Raw KTC values (0–10,000) are remapped so anything below the floor becomes 0, the curve is shaped by the exponent, and the top asset always equals the scale factor. Changing these updates all values instantly — no sync needed.
AdjValue = ScaleFactor × ((RawVal − Floor)^Exp) / ((10000 − Floor)^Exp)
SCALE FACTOR
Output value of the #1 asset. Default 10,000 keeps adjusted values in the same range as raw KTC.
EXPONENT
Curve shape. >1 compresses depth (1.25 = depth matters less). <1 spreads values out.
PLAYER FLOOR
Raw KTC below this = worth 0. Filters out irrelevant depth.
PICK FLOOR
Floor for future picks. Usually slightly lower than player floor.
